# 🎵 Descargador de Audio con yt-dlp

Script en Python para descargar audio desde múltiples URLs (como YouTube) usando yt-dlp.

## 🚀 Cómo funciona
- Lee un archivo `urls.txt` con enlaces (uno por línea)
- Descarga el mejor audio disponible
- Guarda los archivos en la carpeta `Mi_musica`

## 🛠️ Requisitos
- Python 3
- yt-dlp

Instalar:
```bash
pip install yt-dlp
